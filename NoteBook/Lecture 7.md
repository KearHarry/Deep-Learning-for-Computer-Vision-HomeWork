# Lecture 7：卷积神经网络

---

## 📌 概览

本讲引入计算机视觉最核心的架构——卷积神经网络 (CNN)：

1. **卷积层 (Convolution)** — 用滤波器提取局部特征
2. **池化层 (Pooling)** — 降低空间分辨率
3. **批归一化 (Batch Normalization)** — 稳定训练过程

---

## 第一部分：从全连接到卷积

### 1.1 全连接层处理图像的问题

对于一张 $32 \times 32 \times 3$ 的图像（如 CIFAR-10）：

- 展平后维度 $D = 32 \times 32 \times 3 = 3072$
- 如果隐藏层有 $H$ 个神经元，权重矩阵 $W \in \mathbb{R}^{3072 \times H}$

| 问题 | 说明 |
| ---- | ---- |
| 参数爆炸 | 真实图像 $224 \times 224 \times 3 = 150528$ 维，参数量巨大 |
| 丢失空间结构 | 展平后像素的空间关系（上下左右邻居）完全丢失 |
| 不具有平移不变性 | 同一个物体出现在不同位置，全连接层视为完全不同的模式 |

### 1.2 卷积的核心思想（⭐ 关键直觉）

> **不需要看整张图，只需要看局部小区域**，然后在整张图上**滑动**这个"小窗口"。

这就是卷积操作：一个小的**滤波器 (Filter/Kernel)** 在输入上滑动，每个位置做点积，输出一个值。

## 第二部分：卷积层 (Convolution Layer)

### 2.1 卷积操作

**输入**：$C_{\text{in}} \times H \times W$（通道数 × 高 × 宽）

**滤波器**：$C_{\text{in}} \times K_H \times K_W$（必须与输入通道数相同）

**输出**：滤波器在每个位置与输入的对应区域做**元素逐一相乘后求和**，得到一个标量。

$$\text{output}(i, j) = \sum_{c=1}^{C_{\text{in}}} \sum_{m=0}^{K_H-1} \sum_{n=0}^{K_W-1} \text{filter}(c, m, n) \cdot \text{input}(c, i+m, j+n) + b$$

### 2.2 多个滤波器 → 多个通道

- 一个滤波器产生一个**特征图 (Feature Map)**
- $C_{\text{out}}$ 个滤波器产生 $C_{\text{out}}$ 个特征图
- 输出形状：$C_{\text{out}} \times H' \times W'$

### 2.3 输出尺寸公式（⭐ 必须掌握）

$$H_{\text{out}} = \frac{H_{\text{in}} - K + 2P}{S} + 1$$

| 符号 | 含义 | 常见值 |
| ---- | ---- | ------ |
| $K$ | 滤波器大小 (Kernel Size) | 3, 5, 7 |
| $P$ | 零填充 (Padding) | 通常 $P = (K-1)/2$ 保持尺寸不变 |
| $S$ | 步长 (Stride) | 1（不缩小）或 2（缩小一半） |

> **经典配置**：$K=3, P=1, S=1$ → 输出尺寸 = 输入尺寸（"same padding"）

### 2.4 卷积的三大优势（⭐ 对比全连接层）

| 优势 | 说明 |
| ---- | ---- |
| **局部连接 (Local Connectivity)** | 每个输出只依赖于输入的一个小区域，大大减少参数 |
| **权重共享 (Weight Sharing)** | 同一个滤波器在所有位置使用相同的参数 |
| **平移等变 (Translation Equivariance)** | 输入平移 → 输出也相应平移 |

**参数量对比**（$32 \times 32 \times 3$ 输入，输出 $6$ 个特征图）：

- 全连接：$3072 \times 6 = 18432$ 参数
- 卷积 ($5 \times 5$ 滤波器)：$5 \times 5 \times 3 \times 6 + 6 = 456$ 参数（减少 40 倍！）

### 2.5 滤波器学到了什么？

| 网络层级 | 学到的特征 |
| -------- | ---------- |
| 浅层（第一层） | 边缘、颜色、纹理等低级特征 |
| 中层 | 角、轮廓、纹理组合等中级特征 |
| 深层 | 物体部件（车轮、眼睛）等高级特征 |

> 这与 Hubel & Wiesel 发现的生物视觉层级结构惊人地相似！

### 2.6 $1 \times 1$ 卷积

虽然空间范围只有 1 个像素，但在**通道维度**上做全连接：

- 输入 $(C_{\text{in}}, H, W)$ → 输出 $(C_{\text{out}}, H, W)$
- 作用：改变通道数、增加非线性
- 参数量：$C_{\text{in}} \times C_{\text{out}}$

## 第三部分：池化层 (Pooling Layer)

### 3.1 目的

降低特征图的空间分辨率，减少计算量和参数量，增大感受野。

### 3.2 Max Pooling（最常用）

在每个小区域内取**最大值**：

```
输入 (4×4):                输出 (2×2, kernel=2, stride=2):
[1  3  2  4]
[5  6  1  2]     →         [6  4]
[3  2  7  8]               [3  8]
[1  0  3  2]
```

### 3.3 Average Pooling

在每个小区域内取**平均值**。常用于网络最后一层（Global Average Pooling）。

### 3.4 池化 vs 步长卷积

现代网络趋势是用 **stride=2 的卷积**代替池化层，因为卷积是可学习的，而池化是固定操作。

## 第四部分：批归一化 (Batch Normalization)（⭐⭐ 重要）

### 4.1 动机

深度网络训练中，每一层输入的分布会随着前面层参数的更新而变化（**内部协变量偏移**）。这使得训练不稳定。

### 4.2 BN 的操作

对每个特征维度（通道），在一个 mini-batch 内做归一化：

**Step 1**：计算 batch 内的均值和方差

$$\mu_c = \frac{1}{N} \sum_{i=1}^N x_{i,c}$$

$$\sigma_c^2 = \frac{1}{N} \sum_{i=1}^N (x_{i,c} - \mu_c)^2$$

**Step 2**：归一化

$$\hat{x}_{i,c} = \frac{x_{i,c} - \mu_c}{\sqrt{\sigma_c^2 + \epsilon}}$$

**Step 3**：缩放和平移（可学习参数）

$$y_{i,c} = \gamma_c \hat{x}_{i,c} + \beta_c$$

- $\gamma$ 和 $\beta$ 是**可学习**的参数
- 它们允许网络在需要时"撤销"归一化

### 4.3 训练 vs 测试（⭐ 易混淆）

| 阶段 | 均值/方差来源 |
| ---- | ------------- |
| 训练 | 当前 mini-batch 的统计量 |
| 测试 | 训练过程中累积的**指数滑动平均** (running mean/var) |

> 测试时没有 batch 的概念，所以必须用训练时记录的统计量。

### 4.4 BN 在 CNN 中的位置

```
Conv → BN → ReLU → Conv → BN → ReLU → ...
```

对 CNN 来说，BN 是按**通道**归一化的：每个通道有一组 $(\gamma, \beta)$。

### 4.5 BN 的好处

| 好处 | 说明 |
| ---- | ---- |
| 训练更稳定 | 缓解梯度消失/爆炸 |
| 允许更大学习率 | 加速收敛 |
| 正则化效果 | batch 统计量引入噪声，类似 dropout |
| 对初始化不敏感 | 减少了对权重初始化的依赖 |

---

## 第五部分：经典 CNN 结构

### 5.1 典型的 CNN 架构

```
[Conv → BN → ReLU → Pool] × N → [FC → ReLU] × M → [FC → Softmax]
   特征提取部分                      分类部分
```

- 随着网络加深：空间尺寸逐渐**减小**，通道数逐渐**增加**
- 感受野（每个神经元"看到"的输入区域）逐层增大

---

## 💡 重点总结

1. **卷积核心**：局部连接 + 权重共享 + 平移等变，大幅减少参数
2. **输出尺寸公式**：$H_{\text{out}} = \frac{H - K + 2P}{S} + 1$，必须掌握
3. **$K=3, P=1, S=1$**：保持尺寸不变的经典配置
4. **层级特征**：浅层学边缘，深层学语义，类似生物视觉系统
5. **Batch Normalization**：训练用 batch 统计量，测试用滑动平均，按通道归一化
6. **池化层**：降低分辨率，现代网络倾向用 stride=2 卷积替代
7. **CNN 设计范式**：空间尺寸递减、通道数递增
