## 第一部分：线性分类器的局限

### 1.1 线性不可分问题

线性分类器只能学习线性决策边界。以下情况它**完全无法处理**：

| 问题 | 描述 |
| ---- | ---- |
| XOR 问题 | 四个点，对角线相同类别，线性不可分 |
| 环形数据 | 一类在中心，另一类围绕它，无法用直线分开 |
| 多模态类别 | 同一类有多种外观（如不同颜色的汽车） |

### 1.2 解决方案：特征变换（⭐ 关键思想）

如果原始空间中数据线性不可分，我们可以先把数据**变换到另一个空间**，在新空间中它们可能就线性可分了。

**例子 —— 极坐标变换**：

环形数据在笛卡尔坐标 $(x_1, x_2)$ 下线性不可分，但变换到极坐标 $(r, \theta)$ 后，只需要在 $r$ 方向画一条线就能分开。

$$\phi(x) = \begin{bmatrix} r \\ \theta \end{bmatrix} = \begin{bmatrix} \sqrt{x_1^2 + x_2^2} \\ \arctan(x_2 / x_1) \end{bmatrix}$$

> **核心思想**：线性分类器 + 好的特征变换 = 强大的分类器

## 第二部分：神经网络的构建

### 2.1 两层神经网络（⭐ 最基本的形式）

$$f(x) = W_2 \cdot \max(0, \; W_1 x + b_1) + b_2$$

分解来看：

```
输入 x        (D 维)
    ↓
线性变换       h = W₁x + b₁        (H 维)
    ↓
非线性激活     a = max(0, h)        (H 维)  ← ReLU
    ↓
线性变换       s = W₂a + b₂        (C 维)
    ↓
输出分数 s    (C 个类别的分数)
```

### 2.2 为什么需要非线性激活？（⭐⭐ 核心问题）

如果去掉激活函数，多层网络退化为一层：

$$f(x) = W_2(W_1 x) = (W_2 W_1) x = W' x$$

多个线性变换的组合还是线性变换！**非线性激活函数是深度网络能表达复杂函数的关键**。

### 2.3 常见激活函数

| 激活函数 | 公式 | 图形特征 |
| -------- | ---- | -------- |
| ReLU | $\max(0, x)$ | 简单高效，最常用 |
| Sigmoid | $\frac{1}{1+e^{-x}}$ | 输出 (0,1)，容易梯度消失 |
| Tanh | $\tanh(x)$ | 输出 (-1,1)，零中心化 |
| Leaky ReLU | $\max(0.01x, x)$ | 解决 ReLU 的"死神经元"问题 |

### 2.4 深层网络

```
x → [FC → ReLU] → [FC → ReLU] → ... → [FC] → scores
     第1层            第2层                 输出层
```

- **宽度 (Width)**：每层的神经元数量 $H$
- **深度 (Depth)**：层数
- 更深的网络理论上可以用更少的参数表达更复杂的函数

## 第三部分：神经网络的"神经元"视角

### 3.1 单个神经元

一个神经元 = 一个线性变换 + 一个激活函数：

$$a = \sigma(w^T x + b)$$

- 输入：来自上一层的信号 $x$
- 权重 $w$：每个输入的重要性
- 偏置 $b$：阈值
- 激活函数 $\sigma$：引入非线性

### 3.2 与生物神经元的类比

| 生物概念 | 人工神经元对应 |
| -------- | -------------- |
| 树突 (Dendrites) | 输入信号 $x$ |
| 细胞体 (Cell Body) | 加权求和 $w^T x + b$ |
| 轴突 (Axon) | 输出 $a = \sigma(\cdot)$ |
| 突触 (Synapse) | 权重 $w$ |

> **注意**：这个类比非常粗糙。真实的生物神经元远比人工神经元复杂。不要过度解读这个比喻。

## 第四部分：通用近似定理（⭐ 重要理论）

### 4.1 定理内容

> 一个只有**一个隐藏层**的神经网络，只要隐藏层的神经元数量**足够多**，就可以以任意精度近似任何连续函数。

### 4.2 直观理解

- 每个 ReLU 神经元可以"折"一下函数
- $H$ 个神经元就能"折" $H$ 次
- 足够多次的折叠可以拟合任何形状

### 4.3 定理的局限

| 方面 | 说明 |
| ---- | ---- |
| 理论 vs 实践 | 定理保证"存在"这样的网络，但没说如何找到它 |
| 宽度 vs 深度 | 可能需要指数级宽的网络，而深度网络用更少参数就能做到 |
| 优化难度 | 非凸优化，SGD 不保证找到全局最优 |

## 第五部分：非凸优化与损失景观

### 5.1 线性分类器的损失是凸的

凸函数 = 只有一个最低点（全局极小值）。梯度下降保证能找到它。

### 5.2 神经网络的损失是非凸的

- 存在多个局部极小值和鞍点
- SGD 不保证找到全局最优
- 但实践中，**SGD + 好的初始化 + 好的学习率** 通常能找到足够好的解

> **经验观察**：在高维空间中，大多数局部极小值的损失值其实和全局极小值差不多。真正的问题不是局部极小值，而是**鞍点**。

---

## 第六部分：空间扭曲的视角

### 6.1 神经网络在做什么？

每一层在做的事情是：**扭曲输入空间**，使得数据变得线性可分。

```
原始空间（线性不可分）
    ↓  第一层变换（扭曲空间）
中间空间
    ↓  第二层变换（继续扭曲）
新空间（线性可分）
    ↓  最后一层线性分类
输出
```

> **直觉**：如果你有一团缠在一起的绳子（不可分），神经网络的每一层就像在拉扯这团绳子，最终把它们理顺成可以一刀切开的状态。

---

## 💡 重点总结

1. **特征变换**：好的特征变换可以让线性不可分的数据变得线性可分
2. **非线性激活**：没有激活函数，多层网络 = 单层网络，深度毫无意义
3. **ReLU**：$\max(0, x)$，简单高效，是默认选择
4. **通用近似定理**：单隐层网络理论上能拟合任何函数，但实践中深层网络更高效
5. **非凸优化**：神经网络的损失景观是非凸的，但 SGD 实践中表现良好
6. **空间扭曲**：每一层在扭曲数据空间，使之逐步线性可分
