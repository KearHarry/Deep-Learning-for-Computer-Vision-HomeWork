## 第一部分：什么是计算机视觉？

### 1.1 核心定义

计算机视觉的目标：**让机器像人一样理解视觉世界**。

具体来说，就是从图像/视频中提取有意义的信息：

- 这张图里有什么？（**图像分类 Image Classification**）
- 物体在哪里？（**目标检测 Object Detection**）
- 每个像素属于什么？（**语义分割 Semantic Segmentation**）
- 图片里在发生什么？（**图像描述 Image Captioning**）

### 1.2 为什么计算机视觉很难？

对人类来说"看"是毫不费力的，但对计算机来说，一张图片就是一堆数字矩阵：

> 一张 $800 \times 600$ 的彩色图片 = $800 \times 600 \times 3 = 1,440,000$ 个数字

**语义鸿沟 (Semantic Gap)**：像素值与图像语义之间存在巨大的差距。同一个物体在不同条件下，像素值完全不同：

| 挑战 | 说明 |
| ---- | ---- |
| 视角变化 (Viewpoint) | 同一个物体从不同角度看，像素完全不同 |
| 光照变化 (Illumination) | 白天和黑夜同一场景像素差异巨大 |
| 遮挡 (Occlusion) | 物体被部分遮挡，信息不完整 |
| 形变 (Deformation) | 猫可以蜷成一团也可以伸展开来 |
| 背景混杂 (Clutter) | 物体与背景融为一体 |
| 类内差异 (Intraclass variation) | 同一类别（如"椅子"）外观千差万别 |

## 第二部分：计算机视觉简史

### 2.1 生物视觉的启示

- **5.4 亿年前**：寒武纪大爆发，动物开始进化出眼睛。视觉被认为是生物多样性爆发的重要推动力。
- **1959 年 Hubel & Wiesel** 实验：发现猫的视觉皮层中有**简单细胞**和**复杂细胞**，它们对特定方向的边缘有响应。这直接启发了后来的卷积神经网络。

### 2.2 计算机视觉发展里程碑

| 年代 | 事件 | 意义 |
| ---- | ---- | ---- |
| 1966 | MIT "夏季视觉项目" | 最早的 CV 项目，当时以为一个暑假就能搞定 |
| 1970s | David Marr 的视觉理论 | 提出视觉处理的三个阶段：Primal Sketch → 2.5D Sketch → 3D Model |
| 1999 | SIFT 特征 (David Lowe) | 手工设计的局部不变特征，开启了特征工程时代 |
| 2001 | Viola-Jones 人脸检测 | 第一个实时人脸检测算法，用于数码相机 |
| 2009 | ImageNet 数据集发布 | 1400 万张标注图片，1000 个类别 |
| 2012 | AlexNet 赢得 ImageNet 竞赛 | ⭐ **深度学习的转折点**，error rate 从 25% 骤降到 16% |
| 2015 | ResNet | 超越人类水平（error rate 3.6% vs 人类 5.1%） |

### 2.3 深度学习之前 vs 之后（⭐ 重点）

**传统方法**：

```
图像 → [手工特征提取(SIFT/HOG)] → [特征向量] → [分类器(SVM)] → 结果
```

- 特征是人类专家设计的，耗时耗力
- 不同任务需要不同的特征

**深度学习方法**：

```
图像 → [神经网络(端到端学习)] → 结果
```

- 特征是从数据中**自动学习**的
- 同一个框架可以适用于多种任务

> **一句话总结**：深度学习的核心突破是用**数据驱动的特征学习**取代了**人工设计的特征工程**。

### 3.2 数据驱动方法的核心思想（⭐ 重要）

传统编程：

$$\text{程序} = f(\text{规则}, \text{数据}) \rightarrow \text{输出}$$

机器学习：

$$\text{模型} = f(\text{数据}, \text{标签}) \rightarrow \text{规则（权重）}$$

- **不再写死规则**，而是让模型从大量带标签的数据中自己学出规则
- 这就是为什么 ImageNet 这样的大规模数据集如此重要

---

## 💡 重点总结

1. **语义鸿沟**：像素数值与图像含义之间的差距，是 CV 的根本挑战
2. **数据驱动**：不手写规则，让模型从数据中学习
3. **2012 AlexNet**：深度学习在 CV 领域的里程碑，开启了 CNN 主导的时代
4. **端到端学习**：从原始像素直接到最终输出，特征自动学习
5. **Hubel & Wiesel**：生物视觉的层级处理结构直接启发了 CNN 的设计
