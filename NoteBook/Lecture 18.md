## 第一部分：生成模型概述

### 1.1 判别模型 vs 生成模型

| | 判别模型 | 生成模型 |
| -- | -------- | -------- |
| 学习目标 | $P(y|x)$（给定输入，预测标签） | $P(x)$（学习数据分布本身） |
| 能力 | 分类、回归 | 生成新样本、密度估计 |
| 例子 | CNN 分类器、SVM | VAE、GAN、Diffusion |

### 1.2 生成模型的应用

- 图像生成 / 补全 / 超分辨率
- 数据增强
- 异常检测（低概率 = 异常）
- 无监督/半监督学习

## 第二部分：自编码器 (Autoencoder)

### 2.1 基本结构

```
输入 x → 编码器 E → 隐向量 z → 解码器 D → 重建 x̂
损失: L = ||x - x̂||²
```

- **编码器**：将高维数据压缩为低维隐向量 $z$
- **解码器**：从 $z$ 重建原始数据
- **瓶颈**：$z$ 的维度 ≪ 输入维度，迫使网络学习有效压缩

### 2.2 自编码器的局限

- 隐空间 $z$ 没有明确的概率结构
- 无法从隐空间**采样**生成新样本
- 本质上是**压缩/重建**，不是真正的生成模型

## 第三部分：变分自编码器 VAE（⭐⭐ 核心）

### 3.1 核心思想

给自编码器的隐空间**加上概率约束**：编码器不再输出确定的 $z$，而是输出 $z$ 的**分布参数**（均值和方差）。

### 3.2 VAE 结构

```
输入 x
  → 编码器 → μ, log(σ²)     # 输出分布参数
  → 重参数化采样: z = μ + σ · ε, ε ~ N(0, I)
  → 解码器 → 重建 x̂
```

### 3.3 损失函数（⭐ 重要）

$$L = \underbrace{-\mathbb{E}_{z \sim q(z|x)}[\log p(x|z)]}_{\text{重建损失}} + \underbrace{D_{KL}(q(z|x) \| p(z))}_{\text{KL 散度正则化}}$$

**两项的含义**：
1. **重建损失**：让解码器能从 $z$ 重建出 $x$（数据保真度）
2. **KL 散度**：让编码器输出的分布 $q(z|x)$ 接近先验 $p(z) = \mathcal{N}(0, I)$（正则化）

#### KL 散度的解析解

当 $q(z|x) = \mathcal{N}(\mu, \sigma^2 I)$ 且 $p(z) = \mathcal{N}(0, I)$ 时：

$$D_{KL} = -\frac{1}{2}\sum_{j=1}^{d}(1 + \log \sigma_j^2 - \mu_j^2 - \sigma_j^2)$$

### 3.4 重参数化技巧（⭐⭐ 关键）

**问题**：采样操作 $z \sim q(z|x)$ 不可微分，梯度无法反向传播。

**解决**：将随机性"外包"给一个独立的随机变量 $\epsilon$：

$$z = \mu + \sigma \odot \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

这样 $z$ 对 $\mu$ 和 $\sigma$ 是可微的，梯度可以正常传播。

### 3.5 生成新样本

```python
# 训练完成后，直接从先验采样
z = torch.randn(latent_dim)  # 从 N(0, I) 采样
x_generated = decoder(z)      # 解码生成新图像
```

### 3.6 VAE 的特点

| 优点 | 缺点 |
| ---- | ---- |
| 有清晰的概率框架 | 生成图像偏模糊 |
| 隐空间连续、可插值 | 像素级重建损失不够"感知" |
| 训练稳定 | 表达能力受限于高斯假设 |
| 可计算数据似然 | 模式坍塌（但比 GAN 轻） |

---

## 💡 重点总结

1. **自编码器学习压缩**，VAE 学习**概率分布**
2. **VAE 损失 = 重建损失 + KL 散度**：保真度和正则化的平衡
3. **重参数化技巧**是 VAE 可训练的关键——让采样操作变得可微
4. **KL 散度**让隐空间"规整"，使得从 $\mathcal{N}(0, I)$ 采样也能生成合理的图像
5. **VAE 生成的图像偏模糊**，因为像素级重建损失倾向于输出平均值
