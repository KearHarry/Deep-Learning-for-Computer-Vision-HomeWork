## 📌 概览

本讲探讨神经网络的"黑箱"问题——理解网络学到了什么：

1. **特征可视化** — 网络各层看到了什么
2. **显著图 (Saliency Maps)** — 输入哪些像素影响预测
3. **对抗样本** — 微小扰动骗过网络
4. **风格迁移** — 特征可视化的艺术应用

---

## 第一部分：特征可视化

### 1.1 滤波器可视化

**第一层**：直接查看卷积核权重。通常学到的是边缘、颜色、纹理等基本模式。

**高层**：直接看权重没有意义。用优化方法生成最大化某个神经元激活的图像：

$$x^* = \arg\max_x \; f_k(x) - \lambda \|x\|^2$$

- 从随机噪声开始，用梯度上升更新图像像素
- 正则项防止图像变得毫无意义

### 1.2 最大激活 Patch

找到训练集中最大化某个神经元激活的图像区域（patch），直观展示该神经元"关注"什么。

## 第二部分：显著图（⭐ 重点）

### 2.1 梯度显著图

$$\text{Saliency} = \left|\frac{\partial S_c}{\partial x}\right|$$

计算分类分数 $S_c$ 对输入图像每个像素的梯度。梯度大的像素对分类决策影响大。

### 2.2 Guided Backpropagation

在反向传播时，只保留正梯度（对 ReLU 层同时要求前向和反向都为正）：

```python
# 标准 ReLU 反向：mask = (x > 0)
# Guided 反向：mask = (x > 0) AND (dout > 0)
```

生成更清晰的可视化，突出对预测有正面贡献的特征。

### 2.3 Grad-CAM（⭐ 最常用）

```python
# 1. 计算目标类别分数对最后一个卷积层特征图的梯度
alpha_k = global_avg_pool(d_score / d_feature_map_k)

# 2. 加权组合特征图
cam = ReLU(sum(alpha_k * feature_map_k))

# 3. 上采样到输入图像大小
```

- 输出热力图，高亮网络"关注"的区域
- 类别特定的——不同类别的热力图不同
- 不需要修改网络结构

## 第三部分：对抗样本 (Adversarial Examples)

### 3.1 核心发现

在输入图像上添加**人眼不可见的微小扰动**，就能让网络做出完全错误的预测。

$$x_{\text{adv}} = x + \epsilon \cdot \text{sign}(\nabla_x L(x, y))$$

这就是 **FGSM (Fast Gradient Sign Method)**。

### 3.2 意义

- 暴露了神经网络的脆弱性
- 对安全关键应用（自动驾驶、医疗诊断）构成威胁
- 催生了**对抗训练** (Adversarial Training) 等鲁棒性研究

---

## 第四部分：风格迁移 (Style Transfer)

### 4.1 核心思想

分离图像的**内容**和**风格**，将一张图的风格应用到另一张图的内容上。

### 4.2 实现方法

利用预训练 CNN（如 VGG）的不同层特征：

- **内容表示**：高层特征图的激活值
- **风格表示**：各层特征图的 **Gram 矩阵**（通道之间的相关性）

$$G_{ij}^l = \sum_{k} F_{ik}^l F_{jk}^l$$

### 4.3 优化过程

从随机噪声开始，最小化：

$$L = \alpha L_{\text{content}} + \beta L_{\text{style}}$$

- $L_{\text{content}}$：生成图与内容图在高层特征上的 MSE
- $L_{\text{style}}$：生成图与风格图的 Gram 矩阵的 MSE

---

## 💡 重点总结

1. **低层学边缘/纹理，高层学语义/物体**——CNN 的层级特征提取
2. **Grad-CAM** 是最实用的可视化工具，生成类别特定的注意力热力图
3. **对抗样本**揭示了深度网络的脆弱性——微小扰动即可欺骗
4. **风格迁移**利用 Gram 矩阵捕捉纹理/风格信息
5. **可解释性**对于高风险应用至关重要，帮助理解和信任模型决策
